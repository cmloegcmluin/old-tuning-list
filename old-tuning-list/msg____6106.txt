G'day,

Scala1.3 has a wonderful addition which I requested from Manuel Op de Coul
over the past few months. Thanks to him, my students are now using this
Scala command file to try out a musical passage in Csound with any of the
tunings in the Scales archive.

One simply selects the .SCL file from the Scales archive, saves it in the
Scala directory, and then runs the Scala command CS.CMD (which must also
be in the Scala directory). Running this command will generate both Csound
files in their entirety; no manual Csound editing is required at any
stage. The output of this command is two csound files which are ready to
be compiled to produce an audio .WAV file. On a PC (50Mhz 486), the whole
process of selecting a new Scale, making two new Csound files, compiling
them and playing them takes less than three minutes. Under Windows or
Windows95, a single icon allows this command file to be selected, edited,
and run. The output is then compiled and the .WAV file played.

The attached example is suitable for 7-note diatonic tuning but documented
so others can easily make their own modifications. Any 7-note .SCL file
which is loaded into the Scala directory can be accessed from Scala's
scale tables using some of the new lexical functions in the ECHO command
and written straight into a .ORC file. Others may prefer to write tuning
data into the .SCO file or use other functions.

Mark, I think these examples are now ready to leave on your ftp site.
They're my gift to everyone for everything I have got from the tuning
group. And a special thanks to Manuel for a great effort.

Greg S
